POST /categories
Create one category

Creates a category on success; return JSON with status code succes or bad request. Also, returns the created category on success.

Examples

{ 
    'status':{
      'code':200,
      'message':'Success'
    }, 
    'response': {
      'category': {
          'id': 132,
          'name': 'groapa',
          'updated_at': ... ,
          'created_at': ...
        }
    }
}
{ 
    'status':{
      'code': 400,
      'message':'Bad Request'
    }, 
    'response': {
    }
}
