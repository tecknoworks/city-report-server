POST /api/issues
Create one issue with category on the fly

Creates a issue plus a category on succes; return JSON with status code succes or bad request. Also, returns the created issue on success.

Examples

{
    'status':{
      'code':200,
      'message':'Success'
    },
    'response': {
      'issue': {
          'id': 132,
          'title': 'Groapa in fata teatrului din Cluj',
          'latitude': 2.3232,
          'longitude': 21.3232,
          'category_id': 123,
          'updated_at': '2012-09-13 15:14:33',
          'created_at': '2012-09-13 15:06:33'
        }
    }
}
{
    'status':{
      'code': 400,
      'message':'Bad Request'
    },
    'response': {
    }
}
