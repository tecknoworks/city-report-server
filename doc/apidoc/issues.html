>>>

POST /issues
Create one issue

Creates a issue on succes; return JSON with status code succes or bad request. Also, returns the created issue on success.

Supported Formats

json

Errors

400 Bad Request

Examples

{
    'status':{
      'code':200,
      'message':'Success'
    },
    'response': {
      'issue': {
          'id': 132,
          'title': 'Groapa in fata teatrului din Cluj',
          'latitude': 2.3232,
          'longitude': 21.3232,
          'category_id': 123,
          'updated_at': '2012-09-13 15:14:33',
          'created_at': '2012-09-13 15:06:33'
        }
    }
}
{
    'status':{
      'code': 400,
      'message':'Bad Request'
    },
    'response': {
    }
}

Params

Param name Description
title
required

Title of the new issue


Value: Parameter has to be String.
latitude
required

latitude of the new issue


Value: Parameter has to be Float.
longitude
required

longitude of the new issue


Value: Parameter has to be Float.
category_id
required

Issue's category id


Value: Parameter has to be Fixnum.
