POST /api/issues
Create one issue with category on the fly

Creates a issue plus a category on succes; return JSON with status code succes or bad request. Also, returns the created issue on success.

Supported Formats

json

Errors

400 Bad Request

Examples

{
    'status':{
      'code':200,
      'message':'Success'
    },
    'response': {
      'issue': {
          'id': 132,
          'title': 'Groapa in fata teatrului din Cluj',
          'latitude': 2.3232,
          'longitude': 21.3232,
          'category_id': 123,
          'updated_at': '2012-09-13 15:14:33',
          'created_at': '2012-09-13 15:06:33'
        }
    }
}
{
    'status':{
      'code': 400,
      'message':'Bad Request'
    },
    'response': {
    }
}

Params

Param name Description
title
required

Title of the new issue; must be unde issue node


Value: Parameter has to be String.
latitude
required

latitude of the new issue; must be unde issue node


Value: Parameter has to be Float.
longitude
required

longitude of the new issue


Value: Parameter has to be Float.
category_name
required

Issue's category name. WARNING! this parameter must be separate from the issue hash


Value: Parameter has to be String.

PUT api/issue/:id
Edit one issue with creating category on the fly

Edits a issue on succes plus creating a category if needed; return JSON with status code succes or bad request.

Supported Formats

json

Errors

400 Bad Request
404 Not Found

Examples

{
    'status':{
      'code':200,
      'message':'Success'
    },
    'response': {
    }
}
{
    'status':{
      'code': 400,
      'message':'Bad Request'
    },
    'response': {
    }
}
{
    'status':{
      'code':404,
      'message':'Not Found'
    },
    'response': {
    }
}

Params

Param name Description
title
optional

new title; must be unde issue node


Value: Parameter has to be String.
latitude
optional

new latitude; must be unde issue node


Value: Parameter has to be Float.
longitude
optional

new longitude; must be unde issue node


Value: Parameter has to be Float.
category_name
optional

new category name to be created on the fly; must not be under the issue hash(node)


Value: Parameter has to be Fixnum.
