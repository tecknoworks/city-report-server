PUT /category/:id
Edit one category

Also see categories#create.

Edits a category on success; return JSON with status code succes or bad request.

Examples

{ 
    'status':{
      'code':200,
      'message':'Success'
    }, 
    'response': {
    }
}
{ 
    'status':{
      'code': 400,
      'message':'Bad Request'
    }, 
    'response': {
    }
}
{ 
    'status':{
      'code': 404,
      'message':'Not Found'
    }, 
    'response': {
    }
}

Supported Formats

json

Errors

400 Bad Request
404 Bad Request

Params

Param name Description
name
required

Name of the new category


Value: Parameter has to be String.
