POST /categories
Create one category

Also see categories#create.

Creates a category on success; return JSON with status code succes or bad request. Also, returns the created category on success.

Examples

{ 
    'status':{
      'code':200,
      'message':'Success'
    }, 
    'response': {
      'category': {
          'id': 132,
          'name': 'groapa',
          'updated_at': ... ,
          'created_at': ...
        }
    }
}
{ 
    'status':{
      'code': 400,
      'message':'Bad Request'
    }, 
    'response': {
    }
}

Supported Formats

json

Errors

400 Bad Request

Params

Param name Description
name
required

Name of the new category


Value: Parameter has to be String.
