PUT api/issue/:id
Edit one issue with creating category on the fly

Edits a issue on succes plus creating a category if needed; return JSON with status code succes or bad request.

Examples

{
    'status':{
      'code':200,
      'message':'Success'
    },
    'response': {
    }
}
{
    'status':{
      'code': 400,
      'message':'Bad Request'
    },
    'response': {
    }
}
{
    'status':{
      'code':404,
      'message':'Not Found'
    },
    'response': {
    }
}

Supported Formats

json

Errors

400 Bad Request
404 Not Found

Params

Param name Description
title
optional

new title; must be unde issue node


Value: Parameter has to be String.
latitude
optional

new latitude; must be unde issue node


Value: Parameter has to be Float.
longitude
optional

new longitude; must be unde issue node


Value: Parameter has to be Float.
category_name
optional

new category name to be created on the fly; must not be under the issue hash(node)


Value: Parameter has to be Fixnum.
