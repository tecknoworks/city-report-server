>>>

POST /categories
Create one category

Creates a category on succes; return JSON with status code succes or bad request. Also, returns the created category on success.

Supported Formats

json

Errors

400 Bad Request

Examples

{ 
    'status':{
      'code':200,
      'message':'Success'
    }, 
    'response': {
      'category': {
          'id': 132,
          'name': 'groapa',
          'updated_at': '2012-09-13 15:14:33',
          'created_at': '2012-09-13 15:06:33'
        }
    }
}
{ 
    'status':{
      'code': 400,
      'message':'Bad Request'
    }, 
    'response': {
    }
}

Params

Param name Description
name
required

Name of the new category


Value: Parameter has to be String.
